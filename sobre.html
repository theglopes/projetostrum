Sobre o Projeto

O projeto Strum é uma plataforma de venda de jogos digitais desenvolvida pela Aurora Dev LTDA para a Strum LTDA. Este é também um projeto acadêmico no contexto da disciplina de Análise e Projetos de Sistemas.

Nosso objetivo é oferecer acesso fácil aos jogos, compras seguras com métodos de pagamento locais e suporte em português, mantendo uma experiência fluida e moderna.

Tecnologias e Estrutura

  • Frontend: HTML + CSS (tema escuro com acento em roxo).
  • Backend: Java 17, servidor HTTP embutido e JSON via Gson.
  • Banco de Dados: script SQL (MySQL) para esquema inicial.

Status

Em desenvolvimento acadêmico.

Foco

Catálogo de jogos, promoções e experiência simples.
